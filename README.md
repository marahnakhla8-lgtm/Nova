# NovaRP - Next Generation Roleplay Game

A modern, space-themed roleplay game inspired by PEAKRP with a fresh design and enhanced features.

## Features

- **Character Creation & Management**: Create and customize your characters with different classes (Explorer, Merchant, Warrior, Diplomat)
- **Interactive Chat System**: Global chat with simulated other players and real-time messaging
- **World Exploration**: Explore different worlds including Nova Prime, Frontier Worlds, and Urban Centers
- **Character Progression**: Level up your characters through exploration and gain experience points
- **Persistent Game State**: Your characters and chat history are saved locally
- **Modern UI/UX**: Beautiful space-themed design with smooth animations and responsive layout

## Getting Started

1. Open `index.html` in your web browser
2. Click "Start Playing" or "Create Character" to begin
3. Create your first character by filling out the character creation form
4. Select your character and start exploring the Nova universe
5. Use the chat system to interact with other players (simulated)

## Character Classes

- **Explorer**: High agility and intelligence, perfect for discovering new worlds
- **Merchant**: High charisma and intelligence, great for trading and diplomacy
- **Warrior**: High strength and health, ideal for combat situations
- **Diplomat**: High charisma and intelligence, excellent for negotiations

## Game Mechanics

- Characters gain experience through exploration
- Level up to increase your character's stats
- Explore different worlds to discover new content
- Chat with other players in the global chat system
- All progress is automatically saved to your browser's local storage

## Technical Details

- Built with vanilla HTML, CSS, and JavaScript
- Responsive design that works on desktop and mobile
- Local storage for game persistence
- Modern CSS with custom properties and gradients
- Font Awesome icons for enhanced UI

## Browser Compatibility

NovaRP works in all modern browsers that support:
- CSS Grid and Flexbox
- ES6+ JavaScript features
- Local Storage API
